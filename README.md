# OCRUI

set up environment
```bash
virtualenv .ocrenv
```
activate environment
```bash
source .ocrenv/bin/activate
```
install requirements
```bash
pip3 install -r requirements.txt 
```
run application
```bash
python3 run.py
```
stop application
```bash
CTRL+C
```
deactivate environment
```bash
deactivate
```
